# **Noise**  
The continued operation of an Actor’s Capabilities, fulfilling the Actor’s Contract after the Actor’s Purpose is no longer realised in a given System.

---

## 1. A Pedestrian Crossing Button That Still Clicks After the Lights Go Automatic
- **Contract:** click when pressed  
- **Purpose lost:** system no longer uses button input  
- **Behaviour continues:** click still happens  

**Noise:** the click is real, but no longer meaningful to the System.

---

- **Purpose lost:** no light → no signalling  
- **Behaviour continues:** relay keeps clicking  

**Noise:** pattern‑affecting behaviour without realised Purpose.
